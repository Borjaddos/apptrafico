Sobre el Proyecto

¿Qué es este proyecto?

Este proyecto tiene como objetivo visualizar predicciones de transporte público en el País Vasco. A través de un mapa interactivo, los usuarios pueden ver las estaciones de transporte y los datos de ida y vuelta relacionados con las predicciones de ocupación.

¿Por qué se realizó este proyecto?

El proyecto se desarrolló para proporcionar una herramienta fácil de usar que permita a los usuarios entender y anticipar las condiciones de transporte público en tiempo real, ayudando a optimizar el uso de las estaciones de transporte en la región.

¿Cómo funciona?

El proyecto utiliza una base de datos que contiene predicciones de transporte, incluyendo datos de "ida" y "vuelta" en diversas estaciones. Los datos se actualizan en función de la fecha y hora seleccionada por el usuario, y se representan visualmente en un mapa interactivo utilizando Leaflet.js.

Características clave:

  • Visualización de estaciones en un mapa interactivo.
  • Los colores de los marcadores cambian según la cantidad de ida y vuelta.
  • Información detallada de las predicciones por hora.
  • Posibilidad de ver la próxima predicción que supere un umbral predefinido.

Futuras mejoras

Este proyecto está en constante evolución, y se planea agregar más funcionalidades, como predicciones más detalladas y la capacidad de personalizar los umbrales para alertas de ocupación.
